# Reporte LNCS

Archivos:

- `reporte_lncs.tex`: reporte principal en formato LNCS de Springer.
- `referencias.bib`: bibliografia usada por el reporte.

Para compilarlo en Overleaf:

1. Crear un proyecto nuevo.
2. Subir `reporte_lncs.tex`, `referencias.bib` y el archivo oficial `llncs.cls` de Springer si Overleaf no lo detecta automaticamente.
3. Compilar con LaTeX + BibTeX.

En esta computadora no se encontro `pdflatex` ni `bibtex`, por eso no se genero el PDF localmente.
